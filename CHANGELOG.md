# Change Log

All notable changes to the "hadar-icons" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.0.0] - 2026-02-11

### Added
- Initial release of Hadar Icons.
- Vibrant visual hierarchy and customizable folder colors.
- Specialized icon packs for Angular, NestJS, React, Vue, NgRx, Redux, and Vuex.
- Commands to activate theme, change folder colors, configure icon packs, and toggle explorer arrows.
